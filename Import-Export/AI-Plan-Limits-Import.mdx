<Info>
  This is an **administrator** feature for configuring how many AI tokens each
  subscription plan may use per task and model.
</Info>

Upload a spreadsheet to set the default and maximum input/output token limits for each
plan, task type, and model in one step.

<Card title="Sample file" icon="file-arrow-down" href="/files/import-export/openai_plan_limits_sample.xlsx">
  openai_plan_limits_sample.xlsx
</Card>

## How to import

<Steps>
  <Step title="Fill in the spreadsheet">
    One row per (Plan, Task type, Model) combination. Use the sample as a starting point.
  </Step>
  <Step title="Upload">
    Open the AI plan-limits screen and import your file. A summary reports how many rows
    were inserted, updated, and skipped, plus any row errors.
  </Step>
</Steps>

## Columns

| Column | Required | Description | Example |
| --- | --- | --- | --- |
| PlanId | Yes | The plan's numeric id. | `1` |
| TaskType | Yes | The task type's numeric id. | `1` |
| ModelId | Yes | The AI model's numeric id. | `1` |
| IsDefaultModel | No | `Yes`/`No` — is this the default model for the plan + task? | `Yes` |
| DefaultInputToken | Yes | Default input-token allowance. | `1000` |
| DefaultOutputToken | Yes | Default output-token allowance. | `1000` |
| MaxInputToken | No | Maximum input tokens. | `4000` |
| MaxOutputToken | No | Maximum output tokens. | `4000` |

## Limits & rules

| Rule | Value |
| --- | --- |
| File type | `.xlsx` only (contents verified) |
| Maximum size | 5 MB |
| Maximum rows | 5,000 |

<Warning>
  Each `(PlanId, TaskType, ModelId)` may appear only once, and only one row per
  `(PlanId, TaskType)` may be marked `IsDefaultModel = Yes`. The whole file is rejected
  with row details if these rules are broken — nothing is changed.
</Warning>
