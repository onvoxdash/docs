**Data project page**

When you click on a data project from the data project list, you will be navigated to the main page of the selected project. Here, you can view all relevant information about the specific data project.

### **Overview Section**

The **Overview** section provides a comprehensive summary of the data project. This includes key details such as:

* **Collection Date:** The date when the data was collected.  
* **Vendors and Contributors:** Individuals and organizations involved in data collection.  
* **Geographical Regions:** The regions covered by the data project.  
* **Resource Websites:** Links to external resources related to the project.

### **Files Section**

The **Files** section allows you to view and download all available files related to the data project. File types may include raw data, reports, and supplementary documentation.

  <img src="/images/dataproject-files.png" alt="data-project-file-page" style={{ borderRadius: '0.5rem' }} />
</Frame>


### **Questions Section**

In the **Questions** section, you can explore all the survey questions associated with the data project. This section includes filtering options similar to those found in the **Survey Questions** section, enabling you to search and categorize questions efficiently.

</Frame>

### **Metadata Section**

The **Metadata** section provides essential information such as:

* **Collection Date:** When the data was gathered.  
* **Data Project Website:** A link to the project's official website for further details.

  <img src="/images/dataproject-metadata.png" alt="data-project-metadata-page" style={{ borderRadius: '0.5rem' }} />

### **Limitations Section**

Any constraints or restrictions related to the data project are listed in the **Limitations** section. This may include data accuracy, coverage gaps, or methodological limitations.

  <img src="/images/dataproject-limitation.png" alt="data-project-limitation-page" style={{ borderRadius: '0.5rem' }} />

### **Technical Section**

The **Technical** section displays data you have added via the survey data entry page. This includes key statistical details such as:

* **Margin of Error:** The possible deviation in survey results.  
* **Respondent Demographics:** Information about the age groups of participants.  
* **General Population Insights:** Broader demographic data for context.

  <img src="/images/dataproject-techinfo.png" alt="data-project-technicalinfo-page" style={{ borderRadius: '0.5rem' }} />

### **Network Section**

The **Network** section showcases related data projects and dataset connections. This includes:

* **Related Data Projects:** Other projects that share similarities.  
* **Dataset Network:** Common vendors, contributors, and interrelated project insights.
