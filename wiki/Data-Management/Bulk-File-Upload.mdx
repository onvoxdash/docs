The **Bulk New Project Upload** feature allows you to upload an entire folder of files at once, simplifying the data uploading process. Follow these steps to efficiently add your project files:

### **1\. Initiate the Upload**

1. Navigate to **Data Provider \> Files \> Bulk Project Upload**.  
2. Click the blue **Bulk New Project Upload** button located in the top-left corner to select and upload your entire folder.

<Frame>
  <img src="/images/bulk1.JPG" alt="Bulk-Upload" style={{ borderRadius: '0.5rem' }} />
</Frame>

**2\. Select Your File**

1. After the upload is complete, click the blue **Next** button in the bottom-right corner.  
2. This will open the **Upload Files** page. Here, select the uploaded file(s) you want to add to your **Data Projects** in VoxDash.

### **3\. Rename Your Folder (Optional)**

* If needed, click the pencil icon next to the folder name to rename it for better organization and clarity.

<Frame>
  <img src="/images/bulk-modal3.png" alt="Bulk-Upload" style={{ borderRadius: '0.5rem' }} />
</Frame>

### **4\. Finalize the Upload**

* Click the blue **Confirm** button in the bottom-right corner to finalize the upload and make the files visible in the **Data Project** list.

### **Tips for Efficient Uploads**

#### **Uploading Files**

* When uploading a single file, ensure you first create a folder for it. Once the folder is set, change the name and upload the files into it then confirm the changes to complete the process.

### **Requirements**

To successfully use the "Bulk New Project Upload" feature, ensure the following:

* Your folder contains the appropriate SPSS file(s) or other required data formats.  
* File names and folder structures are organized for ease of use.

### **Summary of Steps**

1. Navigate to **Data Provider \> Files \> Bulk Project Upload**.  
2. Click the blue **"Bulk New Project Upload"** button to upload your folder.  
3. Click **Next** to proceed to the **Upload Files** page.  
4. Select your file(s) and add them to your **Data Projects** in VoxDash.  
5. Click **Confirm** to make the uploaded files visible in the project list.
