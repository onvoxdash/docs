# Who this is for?

Use this guide if you primarily consume data and insights — browsing dashboards, searching datasets, exploring survey questions, and viewing maps — without importing or modeling data yourself.

## 1. Discover Relevant Data

- Use [Advanced Search](/Data-Projects/Advanced-Search) to filter by entity, tag, field, date, or text
- Browse the [Projects List](/Data-Projects/Data-Project-List) and pick a project to open its [Project Page](/Data-Projects/Data-Project-Overrview)
- View the [VoxDash Public Archive](/Data-Projects/Voxdash-Public-Archive-Overview),that acts as a central library for freehand public survey data.

## 2. Explore Surveys and Questions

- Open the [Question List](/Survey-Questions/Survey-Questions-List) in a project to see available survey questions
- Drill into a [Question Page](/Survey-Questions/Question-Page-Overview) for distributions, filters, and metadata
- For question‑level mapping, see: [Map Visualization (Questions)](/Maps-&-Visualizations/Map-Visualization-Guide)and [Drilldown Map](/Maps-&-Visualizations/Drilldown-Map-Examples)
- Learn how to view[Viewing Survey Administration Questions](/Survey-Questions/Viewing-Survey-Administration-Questions)

## 3. Understand Vendors and Individuals

- View [Vendors List](/Vendors-&-Individuals/Vendors-Directory) and see vendor's details in [Vendor Page Overrview](/Vendors-&-Individuals/Vendor-Page-Overview)
- View [Individuals List](/Vendors-&-Individuals/Individuals-Directory) and see individual's details in [Individual Page Overrview](/Vendors-&-Individuals/Individual-Page-Overview)
- Understand the diffrences between [Vendors and Individuals](/Vendors-&-Individuals/Vendors-vs-Individuals)

## 4. Organization and Purchaseing New Plans

- Understand [Organization](/Organization-Setup/Understanding-Organization-Public)
- Activate a plan and required add‑ons: [Purchase a Plan](/Billing,-Subscriptions-&-System-Usage/Purchase-Subscription-Plan), [Purchase an Add‑on](/Billing,-Subscriptions-&-System-Usage/Buy-Add--Ons)
