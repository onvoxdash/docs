---
title: "Usage Reports"
---

The **Usage** report category in VoxDash provides a complete overview of how content is created, uploaded, and interacted with across the platform. These reports are valuable for administrators, dataset owners, and compliance teams who need visibility into user activity and asset creation.

Each report helps you track platform engagement, audit user actions, and monitor growth across files, surveys, datasets, profiles, geographic data, and search behavior.



## Uploaded Files
This report lists all **files uploaded** to the platform, including both public and private dataset attachments.

Information included:
- File name  
- File type (e.g., PDF, XLSX, image, CSV)  
- File size  
- Associated dataset name  
- Uploader details (name/email)  
- IP address of the uploader  
- Upload date and time  

Why it’s useful:
- Auditing user uploads  
- Monitoring file growth across datasets  
- Ensuring compliance with data‑sharing policies  
- Identifying large or frequently updated files  



## Uploaded Files Count
Returns the **total number of uploaded files** on the platform.

Use cases:
- High‑level platform usage metrics  
- Monitoring upload trends over time  



## Uploaded Surveys
This report lists all **survey files** uploaded by users, typically in formats like Excel or CSV.

Information included:
- Survey name  
- Sample size (if available in metadata)  
- Associated dataset name  
- Uploader details  
- Public/private status of the resulting dataset  
- Upload date  

Why it matters:
- Tracking new survey additions  
- Monitoring survey volume by teams or organizations  
- Supporting reporting and QA workflows  



## Uploaded Surveys Count
Returns the **total number of surveys** uploaded.

Use cases:
- Survey ingestion tracking  
- Platform growth analytics

- Creation date

- Tracking dataset lifecycle events  



## Datasets Count

Use cases:
- Dataset inventory reporting  
- Tracking dataset creation growth

## Search Terms
This report lists all **search queries** performed by users across VoxDash.

Information included:
- Search term  
- Excluded keywords  
- Applied filters (topics, coverage areas, date ranges)  
- Result counts by category (datasets, questions, vendors, individuals)  
- Search source (web, API, internal tool)  
- User account information  

Why it matters:
- Understanding user information needs  
- Improving dataset discoverability  
- Identifying common topics or gaps in content  
- Supporting analytics and search optimization  

## Search Terms Count
Returns the **total number of searches** performed across the platform.

Use cases:
- Search engagement monitoring  
- Trend detection  
- Platform usability analysis
