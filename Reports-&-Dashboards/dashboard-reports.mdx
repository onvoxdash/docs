
## Trended Questions Count

Returns the **total number of questions** that have been included across all processed trend survey waves.

Use cases:
- Understanding the overall volume of data being trended  
- Providing a high‑level metric for the extent of trend analysis across the platform  
- Benchmarking survey complexity  

## Public Entity Count

Returns the **total number of entities** that have public access enabled.

Use cases:
- Monitoring public exposure  
- Governance and compliance reporting  

## Allowed Domain Count

Returns the **total number of email domains** assigned to the allowlist.

Use cases:
- Governance oversight  
- Tracking domain configuration changes  

## Private Access Count

Returns the **total number of private access permissions** granted within the organization.

Use cases:
- High‑level private access auditing  
- Tracking shifts in internal data‑sharing behavior

## User Group Count

Returns the **total number of user groups** defined in the organization.

Use cases:
- Understanding organizational complexity  
- Measuring access control structure  
- Tracking group creation over time

## Administrator Count

Returns the **total number of administrators** in the organization.

Use cases:
- Tracking admin footprint  
- Ensuring admin access is not overly broad  
- Supporting security audits 
## User Count

Returns the **total number of regular users** registered in the organization.

Use cases:
- Monitoring user base growth  
- Assessing license utilization  
- Reporting on team size

## Organization Count

Returns the **total number of organizations** the current user is a member of.

Use cases:
- Quickly identifying how many workspaces a user belongs to  
- Supporting multi‑organization governance  
- Useful for users who manage multiple departments or client environments
