
## Trended Questions Count

Returns the **total number of questions** that have been included across all processed trend survey waves.

Use cases:
- Understanding the overall volume of data being trended  
- Providing a high‑level metric for the extent of trend analysis across the platform  
- Benchmarking survey complexity  

## Public Entity Count

Use cases:
- Monitoring public exposure  
- Governance and compliance reporting

## Total Page Views Count

A high‑level summary metric that returns the **total number of page views across all entities** for a selected period.

Use cases:
- Platform health monitoring
- Executive‑level reporting
- Traffic trend visualizations

## Individual Views Count

This summary report returns the **total number of views** for individual profile pages.

- Ranking individual contributors by engagement
- Tracking profile growth over time

## Vendor Views Count

A summary report that returns the **total number of views** for each vendor profile.

Useful for:
- Vendor engagement metrics
- Quarterly or annual reporting for partner vendors

## Dataset Views Count

A simplified summary report that returns the **total number of views** for each dataset over a selected time range.

Useful for:
- High‑level performance reporting
- Weekly/monthly analytics summaries
- Comparing dataset interest across categories

## Question Views Count

This report returns the **total number of page views for individual survey questions** within a selected timeframe.

What it measures:
- Total views per question
- Engagement trends across surveys or datasets
- Popularity of specific questions among users

Typical use cases:
- Identifying high‑traffic questions
- Evaluating interest in certain topics or survey sections
- Supporting reporting for research or customer‐insight teams

## Cross‑Tabulation Count

- Engagement with analytical tools inside VoxDash
- Overall usage patterns for the cross‑tab feature

Use cases:
- Monitoring the adoption of advanced analytics features
- Measuring user activity for enterprise usage reports
- Planning resource allocation or performance tuning
